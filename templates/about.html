About This Project

This web application uses a machine learning model to predict whether a breast tumor is malignant or benign based on several medical features. The model is trained on the popular Breast Cancer Wisconsin dataset using scikit-learn and a neural network implemented in TensorFlow/Keras.

Key Features

  • Input form with 30 cancer-related features
  • Model prediction for Malignant or Benign outcome
  • Clean and modern UI built with Tailwind CSS
  • Real-time classification with result highlighting
  • Documentation section explaining all features

Tech Stack

Backend

  • Python (Flask)
  • scikit-learn, TensorFlow/Keras
  • Pandas & NumPy

Frontend

  • HTML5 & Tailwind CSS
  • Font Awesome
  • Jinja2 Templates

Why This Project?

Early detection of breast cancer significantly improves treatment outcomes. With the help of machine learning, healthcare providers can get a second opinion or rapid assessment based on tumor features. This project aims to demonstrate how AI can be used to assist in such medical tasks.
